# Build Threads Clone Application

### Built With React, MongoDB, ExpressJS, NodeJS, JWT, Cloudinary, Socket.io

## Setting .env

```js
PORT=5000
JWT_SECRET=yoursecretjwt
MONGO_URI=mongodb+srv://

CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
```
